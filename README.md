# CommanderAPI
API REST in .NET Core

Implementação de uma REST API em .NET Core 3.1 usando:
  - MVC
  - Repository Pattern
  - Dependency Injection
  - Entity Framework
  - DTO
  - Automapper
  
  
 Implementação baseada no curso: [.NET Core 3.1 MVC REST API - Full Course](https://www.google.com)
